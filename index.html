Creating menus in HTML5

In HTML5 a menu is a list of commands, not just ordinary list elements, but this is still not supported by all browsers (1).

Menus can still be set up like lists:

  • Home
  • Blog
  • About 
        
          <ul>
        <li> <a href="#"> Home </a></li>
        <li> <a href="#"> Blog </a></li>
        <li> <a href="#">About </a></li>
      </ul>

But you can also use the -menu- tag

  • Home
  • Blog
  • About 
    •         
       <menu>
        <li> <a href="#"> Home </a></li>
        <li> <a href="#"> Blog </a></li>
        <li> <a href="#">About </a></li>
 </menu>

    You can also use -menu-item- to label items in your menu - giving you more control and bringing up a pop-up menu (Browser compatibility is limited to Firefox currently): 

          
        <menu>
  
<menu label="Share on...">
    <menuitem label="Twitter" icon="ico_twitter.png"
    onclick="window.open('//twitter.com/intent/tweet?text='+window.location.href);">
    </menuitem>
    <menuitem label="Facebook" icon="ico_facebook.png"
    onclick="window.open('//facebook.com/sharer/sharer.php?u='+window.location.href);">
    </menuitem>
    </menu>

    Global attributes for menu tag

    There are three global attributes for the menu tag:

    • List: A list of comments or elements
    • Context: Must be activated - a pop-up menu
    • Toolbar: Active commands

    As the menu tag is still not functional in many browsers, you can expect the following default settings

            
          menu {
    display: block;
    list-style-type: disc;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 0;
    margin-right: 0;
    padding-left: 40px;
}
